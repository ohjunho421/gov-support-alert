# 정부지원사업 알림 서비스 - SPEC

## 1. 프로젝트 개요

### 1.1 제품명
**Gov Support Alert** - 블로그치트키를 위한 정부지원사업 매칭 알림 서비스

### 1.2 목적
블로그치트키(AI 기반 SEO 최적화 블로그 콘텐츠 생성 SaaS)의 성장을 위해 활용 가능한 정부지원사업을 자동 수집하고, AI가 블로그치트키에 맞는 사업을 매칭하여 텔레그램으로 일 1회 알림을 보냄.

### 1.3 사용자
- **단일 사용자**: 블로그치트키 운영자 (본인)

### 1.4 블로그치트키 프로필 (매칭 기준)
- **서비스 유형**: AI 기반 SaaS, 콘텐츠 마케팅 플랫폼
- **사업 분야**: AI/ML, 콘텐츠테크, 마케팅테크, SEO
- **타겟 고객**: 자영업자, 소상공인, 마케터
- **기술 스택**: React, TypeScript, Node.js, PostgreSQL, Claude API, Gemini API
- **수익 모델**: 구독 기반 SaaS (월 3~10만원)
- **사업 단계**: 초기 스타트업 / MVP 운영 중
- **관련 키워드**: AI, SaaS, 콘텐츠 생성, SEO, 자영업자 디지털 전환, 마케팅 자동화

#### 매칭 가능한 정부지원사업 유형
1. **창업 지원**: 예비창업패키지, 초기창업패키지, 청년창업사관학교
2. **AI/기술 R&D**: AI 바우처, 데이터 바우처, 기술개발사업
3. **소상공인 디지털화**: 스마트상점, 디지털 전환 지원
4. **마케팅/판로**: 온라인 판로 지원, 수출바우처
5. **인력 지원**: 청년 디지털 일자리, 채용 지원금
6. **클라우드/인프라**: 클라우드 바우처, SW 개발 지원

## 3. 핵심 기능

### 3.1 데이터 수집 파이프라인
```
[공공API 호출] + [웹 크롤링] → [데이터 정규화] → [DB 저장] → [중복 제거]
```

### 3.2 AI 매칭 엔진
```
[새 공고 목록] + [블로그치트키 프로필] → Claude API → [관련도 점수 + 요약]
```
- 관련도 점수: 1~10 (7점 이상만 알림)
- AI 요약: "블로그치트키의 [어떤 부분]에 [어떤 지원]을 받을 수 있습니다"
- 신청 조건 요약, 지원 금액, 마감일 포함

### 3.3 텔레그램 알림
- **일 1회** (오전 8시) 새로운 매칭 사업 알림
- 매칭된 사업이 없으면 알림 생략
- 포맷:
```
🏛️ 정부지원사업 알림 (2026.03.18)

📌 [사업명]
💰 지원금액: ~5,000만원
📅 마감일: 2026.04.30
🎯 관련도: 9/10

💡 블로그치트키 매칭 포인트:
AI 기반 SaaS 서비스로 'AI 바우처 지원사업'에 수요기업으로
신청 가능합니다. 기술 고도화 비용을 최대 3억원까지 지원받을 수 있습니다.

🔗 상세보기: [대시보드 링크]
🔗 원문보기: [공고 링크]
```

### 3.4 웹 대시보드
- **전체 공고 목록**: 필터링 (분야, 마감일, 관련도)
- **매칭 결과**: AI 관련도 점수순 정렬
- **공고 상세**: 원문 링크 바로가기
- **마감 캘린더**: 마감일 기준 시각화
- **찜한 공고**: 관심 사업 저장
- **모바일 반응형**

## 4. 기술 아키텍처

### 4.1 기술 스택
| 영역 | 기술 | 이유 |
|------|------|------|
| **프레임워크** | Next.js 15 (App Router) | SSR + API Routes 통합, 대시보드에 최적 |
| **언어** | TypeScript | 타입 안전성 |
| **스타일링** | Tailwind CSS + shadcn/ui | 빠른 UI 개발, 반응형 |
| **DB** | PostgreSQL (Drizzle ORM) | 기존 블로그치트키와 동일 패턴 |
| **크롤링** | Cheerio + Axios | 경량 HTML 파싱 |
| **AI** | Claude API (Haiku 4.5) | 매칭/요약 (비용 효율적) |
| **알림** | Telegram Bot API | 무료, 즉시 설정 |
| **스케줄러** | node-cron | 일일 수집/알림 |
| **배포** | Railway | 기존 인프라 활용 |

## 6. 구현 우선순위 (MVP)

### Phase 1: 핵심 파이프라인
1. 프로젝트 세팅 (Next.js + DB)
2. 기업마당 API 연동 (가장 데이터 많음)
3. AI 매칭 엔진 (Claude Haiku)
4. 텔레그램 봇 알림
5. 크론 스케줄러

### Phase 2: 대시보드
6. 공고 목록 페이지
7. 매칭 결과 페이지 (관련도순)
8. 공고 상세 + 원문 링크
9. 모바일 반응형

### Phase 3: 확장
10. 추가 데이터 소스 (크롤링)
11. 마감 캘린더
12. 찜하기 기능
13. 알림 설정 (임계점 조정)
